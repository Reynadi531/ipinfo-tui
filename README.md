# IPInfo TUI

![image](https://user-images.githubusercontent.com/43875921/209363107-8ef91f0a-11ee-4e06-b01e-3902f80111eb.png)


This is the imporvment of the prodecesor the [IPInfo CLI]("https://github.com/Reynadi531/ipinfo-cli"). The purpose of this app is to make it easily interact with ipinfo from terminal. For those who dont know, ipinfo is ip geolocating platform. Its made with golang and powered by [ipinfo](https://ipinfo.io/)
